# Plannify Shared Libraries

This repository contains shared libraries used across various Plannify projects. These libraries are designed to be modular and reusable, providing common functionality that can be leveraged in different applications.

## Crates

### S3 Manager

A Rust crate for managing interactions with Amazon S3 compatible storage services. It provides functionalities for uploading and downloading documents.

### SMTP Manager

A Rust crate for managing email sending functionalities using SMTP protocol. It supports sending various types of emails.
